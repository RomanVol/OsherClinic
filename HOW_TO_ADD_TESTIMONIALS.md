# הוספת המלצות לאתר הקליניקה של אושר

## איך להוסיף המלצות אמיתיות?

כל ההמלצות נמצאות בקובץ אחד ומרכזי: **`app/data/testimonials.ts`**

### צעדים להוספת המלצות:

1. **פתחי את הקובץ**: `app/data/testimonials.ts`

2. **בחרי את סוג ההמלצה** שאת רוצה לערוך:
   - `homeTestimonials` - המלצות לדף הבית
   - `massageTestimonials` - המלצות לטיפול מאסג'
   - `consciousnessTestimonials` - המלצות לטיפול תודעה
   - `lectureTestimonials` - המלצות להרצאה

3. **הוסיפי או ערכי המלצות** בפורמט הבא:

```typescript
export const massageTestimonials: Testimonial[] = [
  {
    name: "רחל כהן",
    text: "הגעתי לאושר עם כאבי גב כרוניים. הטיפול היה פשוט מדהים והקלה מיידית!",
    service: "טיפול מאסג'"
  },
  {
    name: "שרה לוי", 
    text: "חוויה מרגיעה במיוחד! ממליצה בחום!",
    service: "טיפול מאסג'"
  },
  // הוסיפי כאן עוד המלצות...
];
```

### שדות בכל המלצה:

- **`name`** - שם המטופלת (חובה)
- **`text`** - תוכן ההמלצה (חובה)
- **`service`** - שם הטיפול (אופציונלי)

### דוגמאות מלאות:

#### דוגמה 1: המלצה פשוטה
```typescript
{
  name: "מרים",
  text: "טיפול נפלא! אושר היא מטפלת מקצועית ומדהימה."
}
```

#### דוגמה 2: המלצה עם פרטי טיפול
```typescript
{
  name: "דינה אברהם",
  text: "אחרי חודשים של מתח וכאבים, הטיפול של אושר הביא לי הקלה אמיתית. השמנים האתריים והמגע המקצועי יצרו חוויה מיוחדת.",
  service: "טיפול מאסג' עם שמנים אתריים"
}
```

### טיפים חשובים:

✅ ודאי שכל המלצה מסתיימת בפסיק (`,`) חוץ מהאחרונה
✅ השתמשי במירכאות כפולות (`"`) סביב הטקסט
✅ אפשר להוסיף כמה המלצות שרוצים - לא מוגבל ל-3
✅ ההמלצות יופיעו באופן אוטומטי באתר

### איפה ההמלצות יופיעו?

- **דף הבית** - `homeTestimonials`
- **עמוד טיפול מאסג'** - `massageTestimonials`  
- **עמוד טיפול תודעה** - `consciousnessTestimonials`
- **עמוד הרצאה** - `lectureTestimonials`

---

## פתרון בעיות נפוצות:

**בעיה**: האתר לא עובד אחרי שהוספתי המלצה
**פתרון**: בדקי שכל המירכאות וסימני הפסיק במקום הנכון

**בעיה**: ההמלצה לא מופיעה
**פתרון**: שמרי את הקובץ ורענני את הדפדפן (Cmd/Ctrl + R)
